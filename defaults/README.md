#Github pages hosting

## Step 1

> git init `<blog.repo>`
> cd blog.repo
> git remote -v
> git remote rm origin
> git remote add origin git@github.com:{username}/{repo.name}.git

## Step 2

> yo new `<project name>`
> yo compile `<blog.repo>`

## Step 3

> cd `<blog.repo>`
> git push


Done
